Transana specializes in assisting with transcribing audio and video data. It integrates the media player and word processing environment into one split screen. You can also code documents, perform queries, and create memos and reports. Transana uses a different of names to describe these features than other tools. Libraries are used to organize source files. Collections are used to group snippets of data, these includes clips (video), quotes (text), and snapshots (image). Clips, quotes, and snapshots are coded using keywords. Notes can be used for analytical memos, or simple notes associated with a particular library, document, or collection.
